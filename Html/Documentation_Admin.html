Search API (V1) Overview

This API requires user authentication. You will need an API key to pass with your requests to provide authentication. 

https://www.winkitsearch.com/api.php?module=searchStats&method=getStats&aid=[AID]&key=[KEY]&date_from=[DATE_FROM]&date_to=[DATE_TO]&format=[FORMAT]

Request parameters

AID Affiliate ID. In your case this should be 174 Required
KEY Your API KEY 4b1bf945f0a69def1ceeb38554c374b403e9961f Required
DATE_FROM The start date to fetch report from. Required
DATE_TO The end date to fetch report to. Required
SID Your Feed ID Optional
FORMAT Output format: Optional
json - JSON Format
csv - CSV Format

Response parameters

error_code Return error code
error_msg Return error message
